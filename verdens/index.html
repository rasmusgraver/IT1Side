Dagens challenge

Lag denne nettsiden: https://www.verdensdagen.no/

Det er IKKE lov å:
  • Kommunisere med dine medelever
  • Bruke ChatGPT og lignende
  • Å bruke inspect på verdensdagen.no siden!
(du må gjerne bruke inspect på din egen side mens du jobber)

Noen ting du kan trenge:

  • Fargekodene som er brukt på siden:
    • Body bakgrunnsfarge: #F5E5D3
    • Grønn knapp farge: #107868
    • Blå-Lilla farge: #343CA8 (brukt flere steder)
    • Orange farge: #FA8800
  • transform: rotate(-4deg); - Dette er for å skråstille skriften

logo
3 striper
YouTube-bildet
